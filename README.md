# team6-Daegu-

## 프로젝트명 
대구시 어린이교통사고 위험지역 도출 및 예방책 제시

## 분석 배경 및 목적
2024년 대구시에서 발생한 어린이 교통사고건수가 비수도권 1위를 차지,
이에 따라 어린이교통사고 위험지역을 도출하고 
위험지역 사고예방 시설 설치 및 어린이보호구역 지정 제안
